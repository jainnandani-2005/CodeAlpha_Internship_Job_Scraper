# CodeAlpha Data Analytics Internship Project

## Project Title
Book Data Web scraper

## Description
This Project scrapes book titles and prices from the books to Scrape website using python. The scraped data is stored in a CSV file and analyzed using Pandas and matplotlib.

## Technologies Used
-Python 
-Requests
-BeautifulSoup
-Pandas
-Matplotlib

## Files
-scraper.py - Scrapes the data 
-books.csv - stores the scraped data
-analysis.py - Analyzes the data and creates a graph
-README.md - Project documentation
-requirements.txt - Required Python libraries

## Output
-CSV file containing book titles and prices
-Bar chart showing the prices of the top 10 books
